# 小红书图片生成提示词·通用框架

以下是一套可在任何AI客户端使用的提示词模板，输入后可获得精准的图片生成指令。

---

## 📋 用户输入模板

```markdown
# 任务：为小红书笔记生成配图提示词

## 我的内容主题
[在此粘贴你的小红书笔记内容或大纲]

## 期望风格（选择一项）
- [ ] Second Brain极简风（黑白灰+橙色点缀）
- [ ] Anthropic优雅风（奶油橙紫渐变）
- [ ] 科技未来风（深蓝青色渐变）
- [ ] 温暖生活风（米色粉色系）

## 图片数量
封面图 + [  ] 张内容图

## 特殊要求
- 目标平台：小红书
- 文字：全中文
- 强调点：[数据/案例/对比/教程]
- 其他：___________

---

请基于以上信息，输出：
1. 统领全局的视觉风格模板
2. 每张图片的详细生成提示词（包含布局、文字、颜色、尺寸等）
3. 视觉规范说明

要求：
- 提示词要足够详细，可直接用于生图工具
- 所有图片保持视觉统一性
- 标注清楚文字内容、位置、字号、颜色
- 用符号和表格描绘布局结构
```

## 🔧 使用示例

### 示例输入

```markdown
# 任务：为小红书笔记生成配图提示词

## 我的内容主题
标题：AI提示词工程师月入3W的秘密
内容大纲：
1. 什么是提示词工程师
2. 3个必备技能
3. 5个实战案例  
4. 月薪数据对比
5. 学习路线图

## 期望风格
- [x] Second Brain极简风

## 图片数量  
封面图 + 5 张内容图

## 特殊要求
- 强调点：数据对比、技能清单
- 吸引点击，年轻化
```

### AI应输出

按照上述"AI输出标准框架"，生成：

1. Second Brain风格的完整视觉规范
2. 1张封面图的详细提示词（含"月入3W"等吸引眼球的数据）
3. 5张内容图的详细提示词：
   - 图1：职业定义对比图
   - 图2：3技能卡片式布局
   - 图3：5案例时间轴展示
   - 图4：薪资对比表格
   - 图5：学习路线流程图
4. 视觉一致性规范

## 💡 关键成功要素

**让AI输出精准提示词的秘诀：**

1. **结构化输入**：用清晰的markdown格式组织需求
2. **参数明确**：指定风格、数量、强调点
3. **示例引导**：必要时提供参考图片或案例
4. **细节要求**：明确提出文字、颜色、尺寸需求
5. **迭代优化**：第一次输出后，针对性调整

**AI应输出的提示词特点：**

1. **可视化描述**：用符号、表格、框线模拟布局
2. **参数完整**：包含颜色代码、尺寸数值、字号字重
3. **层级清晰**：标注每个元素的位置、大小、关系
4. **文字精确**：写明所有需要出现的文字内容
5. **风格统一**：遵循全局视觉规范DNA
